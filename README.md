Laravel Auth app is simple Register and login form for user.

## Project Setup

```
git clone git@github.com:adnansiddiq/laravel-auth.git
cd laravel-auth
composer install
./vendor/bin/sail up
./vendor/bin/sail shell
php artisan migrate 
npm install
npm run dev 
```


## Demo links

- Register : http://localhost/register
- Login : http://localhost/login
